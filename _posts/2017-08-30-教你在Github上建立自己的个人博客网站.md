# 概述
之前闲着没事,就利用Github建了一个个人博客网站,效果还不错,今天就来分享一下.

1.面试装逼,这个不必多说...
2.把平时积累的知识和项目记录下来,方便日后查看使用
3.不受其他博客平台的限制
